-  Reporting PDF
-  Suggestions intelligent for improving the performance system


## How to run project
- Clone this repo `
    git clone https://github.com/luuthuong/e-smart-iot.git
`
<br/>
-  `
    cd e-smart-iot
`
- Install dependencies packages: `
    npm install
`
- Run with development env: `
  npm run dev
`
- To build production: `
    npm run build
`
